Description
-
This project provides a simple example of how to embed an OpenGL renderer into a Tcl/Tk frame. 
Please note that this code was extracted from an old project, so it may be a bit messy. Thank you for your understanding.

Test Environment
-
* MSYS2 + MinGW64
* Tcl/Tk 8.6
